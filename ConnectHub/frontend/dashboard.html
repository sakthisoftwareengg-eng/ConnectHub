System Information

ConnectHub runs in Hybrid Mode. When online, messages transmit instantly via real-time WebSockets.

When offline, the app switches to Local Buffer mode. Messages are queued locally inside IndexedDB.

Tip: Click the connection widget in the navbar to toggle offline/online status to test the sync queue!
